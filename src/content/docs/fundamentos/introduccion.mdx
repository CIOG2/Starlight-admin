## ¿Qué es Destiny?
Destiny, es un sistema de diseño multimarca creado por el equipo de diseño de PriceTravel Holding. Incluye una guía detallada sobre la experiencia de usuario y cómo implementar de forma adecuada componentes en una interfaz de usuario en múltiples plataformas digitales: **Web, Mailing & Apps**.


## Todo sobre Destiny
Un **sistema de diseño** es una colección de recursos prediseñados y reutilizables (componentes, patrones, guías y código) que permite a sus usuarios crear experiencias digitales consistentes más rápido. 

Al utilizar los recursos prediseñados y universales de **Destiny**, se minimiza el tiempo que los equipos dedican a diseñar y crear. En lugar de crear y reconstruir elementos básicos, pueden dedicar ese tiempo a personalizar sus productos para abordar casos de uso específicos de los clientes.



## 💫 Enfoque Multimarca
Destiny está construido con un enfoque **Multimarca**, lo que significa que permite gestionar de manera eficiente el diseño de múltiples marcas dentro de una misma organización o grupo. 

Está diseñado para ofrecer consistencia visual y funcionalidad compartida entre diferentes identidades de marca, manteniendo a la vez la flexibilidad necesaria para que cada marca conserve su propia personalidad y características distintivas.



## Características principales de un Multibrand Design System:
1. **Componentes reutilizables con variantes específicas para cada marca**:
   Los componentes de interfaz de usuario (botones, formularios, íconos, etc.) están diseñados para ser reutilizados por varias marcas, pero con opciones para personalizarlos según los lineamientos de cada una.

2. **Estilos de marca personalizados**:
   Cada marca dentro del sistema puede tener su propio set de colores, tipografía, logotipo y otros elementos visuales, mientras que los componentes y la estructura básica se mantienen consistentes.

3. **Eficiencia en el desarrollo**:
   Al tener un sistema compartido, los equipos de diseño y desarrollo pueden trabajar de manera más eficiente. No necesitan crear todo desde cero para cada marca; simplemente adaptan los componentes ya existentes a las necesidades específicas de cada marca.

4. **Escalabilidad**:
   Un sistema multibrand permite escalar el diseño y el desarrollo de manera ágil a través de las diferentes líneas de productos y marcas que tiene PriceTravel Holding.

5. **Guías de diseño compartidas**:
   Las guías de diseño (design guidelines) aplican a todas las marcas, pero también especificaciones para cada una. Esto facilita que diferentes equipos sigan una estructura básica mientras adaptan el diseño a la identidad de cada marca.



## Cómo funciona Destiny
El sistema de diseño está organizado en tres partes principales:

- **Fundamentos**: Son los elementos visuales necesarios para crear experiencias de usuario atractivas de principio a fin. Esto incluye orientación sobre iconografía, tipografía, diseño y estructura.
- **Componentes**: Son los bloques interactivos reutilizables que permiten crear una interfaz de usuario (UI).
- **Patrones**: Los patrones son soluciones recomendadas para que un usuario logre un objetivo. Muestran combinaciones reutilizables de componentes y plantillas que abordan objetivos comunes de los usuarios con secuencias y flujos.
